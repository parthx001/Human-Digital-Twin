# Recommendations Tab - Smart Decision Support

## Overview

The **Recommendations** tab provides intelligent, actionable suggestions based on your simulation results. It analyzes your input parameters and output metrics to offer targeted advice for improving student outcomes and reducing dropout risk.

## How It Works

The suggestions are automatically generated by analyzing:

1. **Stress Levels** - High stress triggers stress management recommendations
2. **Dropout Risk** - High dropout risk triggers urgent interventions
3. **Engagement Metrics** - Individual category performance identifies specific improvement areas
4. **Academic Performance** - CGPA and related metrics suggest support services

## Recommendation Types

### 🚨 High Priority (Red/Orange)
- **High Stress Level** (>80%): Reduce workload or increase break times
- **High Dropout Risk** (>50%): Implement immediate intervention programs
- **Low Attendance** (<75%): Set attendance goals and track progress
- **Insufficient Assignments** (<60% completion): Create schedules and reminders
- **Inadequate Sleep** (<75% of 8 hours): Establish consistent sleep schedule

### 🟡 Medium Priority (Orange)
- **Below-Target CGPA** (<70%): Seek tutoring or academic support

### ✨ Positive Feedback (Green)
- **Good Performance**: Maintain current habits and continue progressing
- When stress is low (<40%) AND dropout risk is low (<30%)

## Features

### Smart Analysis
- Analyzes all engagement factors (attendance, assignments, sleep, CGPA)
- Calculates risk indicators (stress, dropout probability)
- Generates contextual recommendations

### Priority Indicators
- **Red border** = High priority, urgent action needed
- **Orange border** = Medium priority, improve soon
- **Green border** = Positive feedback, continue current path

### Detailed Information
Each suggestion includes:
- **Title**: What the issue is
- **Action**: What to do about it
- **Detail**: Metric-specific information (e.g., "Current: 6 hours/night (Target: 8+ hours)")

### Real-time Updates
- Suggestions update automatically when you run a new simulation
- No manual refresh needed
- Shows cumulative impact of all factors

## Technical Details

### Scoring Thresholds

- **Attendance**: <75% triggers recommendation
- **Assignments**: <60% of max (14/week) triggers recommendation
- **Sleep**: <75% of ideal (8 hours) triggers recommendation
- **CGPA**: <70% of max (10) triggers recommendation
- **Stress**: >80% triggers high-priority recommendation
- **Dropout Risk**: >50% triggers high-priority recommendation

### Calculation Methods

```
Attendance Score = input_attendance (0-100)
Assignment Score = min(100, (assignments / 14) * 100)
Sleep Score = min(100, (sleep / 8) * 100)
Academic Score = (cgpa / 10) * 100
```

## Limitations

- Recommendations are based on input parameters, not actual outcomes
- Generic action items (customize based on institution)
- Threshold values can be adjusted per institution
- Doesn't account for external factors (family emergencies, health issues, etc.)
